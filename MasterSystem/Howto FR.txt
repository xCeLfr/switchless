La console intègre un mod 50Hz ou 60Hz.
Toutes les informations sur l'installation ici : https://github.com/xCeLfr/switchless
Le mode choisi est sauvegardé dans la puce.

Notice :
Le changement de mode s'effectue console allumée avec un appui long( plus de 3sec) sur le bouton PAUSE.
- Appui long sur le bouton PAUSE pour changer de mode.
- Appui court pour mettre pause. 
- Une led indique l'état (vert = 50Hz, Bleu = 60Hz)
